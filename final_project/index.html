5 Calculate Potential Solar Contribution

I performed the following calculations in the calculate_power_panel() function, which, by passing in the solar flux, ambient temperature and wind speed values averaged over the area of each state, enabled me to calculate the average AC power output of a single panel in each state.

To calculate the contribution from solar panels, I calculated the cell temperature, \(T_c\), that each solar panel in a given state would be exposed to using Equation 1, which is taken from the course text, @jacobson100CleanRenewable2020a Eq 5.9. Unless otherwise stated, all equations are referenced from the course text. \[ T_c = T_a + 0.32 * (F_{cur}/(8.91 + 2*w)) \tag{1}\]

I then computed \(C_temp\), the correction for the cell temperature Equation 2, \[ C_{temp} = 1 - b_{ref} * \max\{ \min \{T_c - T_{ref}, T_{th} \}, 0 \} \tag{2}\]

The AC power, \(P_ac\), from each cell was given by Equation 3.
\[ P_{ac} = F_{cur} * A_{panel} * E_{panel} * C_{temp} * D_f \tag{3}\]

I used a standard values taken from the text for all constant values, see Table 1.

6 Calculate Potential Wind Contribution

I performed the following calculations in the calculate_power_turbine() function. I passed in the average wind speed in a given state \(V_m\), and calculated the potential wind power to be generated from a single turbine exposed to the ambient conditions in that state.

I assumed standard constant values for the air density, \(ρ = 1.23 \si{kg/m^3}\), and the diameter of the wind turbine \(D_t = 127 \si{m}\). I calculated the swept area of the turbine, \(A_t\) (m^2), using Equation 4 (Eq 6.7). \[ A_t = π * D_t^2 / 4 \tag{4}\]

I then calculated the power available in the wind, \(P_m\) (W), using the average wind speed in a given state, \(V_m\) and Equation 5 (Eq 6.11) \[ P_m = (6/ π) * (0.5) * rho * A_t * V_m^3 \tag{5}\]

Once I had \(P_m\) in each state, I determined what would be an appropriate power rating, \(P_r\) for a turbine in that state by rounding the value of \(P_m\) up to the next 500. I then calculated the capacity factor for a turbine with this rating using Equation 6 (Eq 6.28). \[ CF = 0.087 * V_m - ((P_r)/(D_t^2)) \tag{6}\]

I then calculated the expected power from a single turbine in a given state by multiplying the capacity factor by the rated power in Equation 7 (Eq. 6.24). \[ P_t = P_r * CF \tag{7}\]

I accounted for losses that would occur in the system due to curtailment using the following equations, Equation 8 and Equation 9 (Eq 6.31). The values of the losses I used are described in Table 2. \[ L_t = (1 - (p_{L_td} * p_{L_d} * p_{L_c} * p_{L_a} )) \tag{8}\]

\[ P_{t \ after \ losses} = P_t * (1 - L_t) \tag{9}\]
